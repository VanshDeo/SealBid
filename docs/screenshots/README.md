# SealBid Documentation Screenshots

This directory stores screenshot proof artifacts required for the Midnight DApp submission checklist.

## Required Screenshots

1. **`compact-compile.png`** (or `.jpg`):
   - **Description**: Screenshot of successful `compactc` / `npm run compile:compact` terminal output listing the 3 generated circuits (`submit_sealed_bid`, `reveal_bid`, `settle_auction`).
   - **Path**: `docs/screenshots/compact-compile.png`

2. **`contract-deployed.png`** (or `.jpg`):
   - **Description**: Screenshot showing successful smart contract deployment to Midnight Preview/Preprod testnet with the visible contract address `0xcontract_sealbid_preview_7f3a9b1c2e4d5f`.
   - **Path**: `docs/screenshots/contract-deployed.png`

Place your image files in this folder or reference your public image URLs in `README.md`.
